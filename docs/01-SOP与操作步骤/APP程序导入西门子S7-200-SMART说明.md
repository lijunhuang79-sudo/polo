# 如何将本 APP 生成的程序用于西门子 S7-200 SMART

**文档版本**：1.0  
**适用**：PLC 编程仿真器（本 Web 应用）生成的 STL/LAD/SCL 代码 → 西门子 S7-200 SMART 实机  

---

## 1. 本应用能提供什么

- 在本站生成**控制方案**后，你会得到：
  - **STL**（语句表）、**LAD**（梯形图文本描述）、**SCL**（结构化文本）三种形式的程序**文本**；
  - 一份 **HTML 导出报告**（「导出方案」按钮），内含上述代码、I/O 表、BOM 等。

- 本应用**不能**直接生成：
  - 西门子 STEP 7-Micro/WIN SMART 的**工程文件**（`.smart` 等）；
  - 也不能被 S7-200 SMART 或 TIA Portal **一键导入**。

因此，要把“APP 里的程序”用到 S7-200 SMART，需要**在西门子软件里按本应用的代码重新做一遍程序**，再下载到 CPU。

---

## 2. 推荐流程：从本 APP 到 S7-200 SMART 实机

### 步骤 1：在本 APP 中生成并导出

1. 在本站选择场景（如「启保停」「延时启动」「星三角」等），生成方案。
2. 在 **STL / LAD / SCL** 标签页中确认逻辑和地址（如 I0.0、I0.1、Q0.0、T37 等）。
3. 点击 **「导出方案」**，保存 `PLC_Solution_xxxx-xx-xx.html`，便于随时查看代码和 I/O 表。

### 步骤 2：在西门子软件中新建项目并录入程序

1. 安装 **STEP 7-Micro/WIN SMART**（西门子 S7-200 SMART 官方编程软件）。
2. 新建项目，选择与实际一致的 **CPU 型号**（如 CPU SR20、ST30、CR40 等）。
3. 打开**程序块**（如 MAIN），用本 APP 的代码作为**参考**，在软件中录入程序：
   - **推荐**：在 STEP 7-Micro/WIN SMART 中打开 **LAD** 或 **STL** 视图，按本 APP 的 LAD 说明或 STL 语句**逐网络手工输入**；
   - 本 APP 的 STL 指令（如 `LD`、`O`、`AN`、`=`、`TON` 等）与 S7-200 SMART 的指令集**基本一致**，可直接对照录入，注意定时器/计数器号（如 T37）与 PLC 中的资源一致。
4. 根据本 APP 的 **I/O 表** 在项目中配置/核对**符号表**（可选），保证地址与现场接线一致（如 I0.0=启动、I0.1=停止、Q0.0=接触器等）。

### 步骤 3：下载到 S7-200 SMART CPU

1. 用 **网线**（Ethernet）或 **USB** 连接编程电脑与 S7-200 SMART CPU。
2. 在 STEP 7-Micro/WIN SMART 中设置 **PG/PC 接口**（通信方式、网口或 USB）。
3. 使用 **「下载」** 将程序块下载到 CPU（可勾选“程序块”“数据块”等）。
4. 将 CPU 切换到 **RUN**，在现场根据 I/O 表接线后测试。

---

## 3. 注意事项

| 项目 | 说明 |
|------|------|
| 指令与地址 | 本 APP 生成的 STL/LAD 与 S7-200 SMART 指令集高度相似，但定时器/计数器编号、数量因 CPU 型号而异，需在西门子软件中确认并保持一致。 |
| 无自动导入 | 西门子工程为专用格式，本 APP 不提供也不支持“导入到 STEP 7-Micro/WIN SMART”的自动转换，只能以“参考文本 + 手工录入”方式使用。 |
| 导出报告 | 导出的 HTML 仅用于**查阅与存档**，不能替代在 STEP 7-Micro/WIN SMART 中建立的工程。 |

---

## 4. 小结

- **本 APP**：用于方案设计、逻辑仿真和生成可读的 STL/LAD/SCL 文本及导出报告。  
- **西门子 STEP 7-Micro/WIN SMART**：用于在 S7-200 SMART 上建立工程、录入程序并下载到 CPU。  
- **实际操作**：以本 APP 的代码和 I/O 表为参考，在 STEP 7-Micro/WIN SMART 中手工录入（或按 LAD 搭接），再下载到 S7-200 SMART 即可。

如需不同 CPU 型号的地址与资源说明，请参考西门子 S7-200 SMART 系统手册与 STEP 7-Micro/WIN SMART 帮助。
